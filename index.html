Basic Commands with grep

grep -n "word" file.txt     # Show line numbers
grep -c "word" file.txt     # Count matches
grep -v "word" file.txt     # Invert match
grep -P "pattern" file.txt  # Perl regex features

Regex Fundamentals

.Any single character
^Beginning of a line
$End of a line
*0 or more occurrences
+1 or more occurrences
?0 or 1 occurrence
|OR
()Grouping

Character Classes

[abc]a, b, or c
[a-z]lowercase letters
[A-Z]uppercase letters
[0-9]digits
[^0-9]non-digits

ðŸ§  Try It:

  • Letter followed by digit: [A-Za-z][0-9]
  • 3 digits in a row: [0-9]{3}
  • Non-letter at start: ^[^A-Za-z]

POSIX Character Classes

grep '[[:digit:]]' file.txt
grep '[^[:alpha:]]' file.txt
grep '^[[:upper:]]' file.txt

Tip: These are portable across systems and useful in shell scripts.

Perl-Compatible Shortcuts

\dDigit
\DNon-digit
\wWord character
\WNon-word character
\sWhitespace
\SNon-whitespace
